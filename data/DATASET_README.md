
# Dataset Information

This project uses the:

UCI Default of Credit Card Clients Dataset

Source:
https://archive.ics.uci.edu/ml/datasets/default+of+credit+card+clients

The dataset is automatically downloaded through OpenML when running:

python run_all.py

After downloading, the dataset will automatically be saved as:

data/uci_credit_card_default.csv

This approach improves reproducibility and ensures researchers always use the same public dataset source.

The dataset contains:
- repayment history
- credit limit information
- bill statements
- payment amounts
- borrower demographics
- default labels

Research Areas:
- credit risk modeling
- financial machine learning
- robust AI
- decision systems
- fairness-aware ML
