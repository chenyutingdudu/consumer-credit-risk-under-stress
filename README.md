# Consumer Credit Risk Under Stress

Research project exploring predictive bias, calibration, and decision risk under economic stress conditions.

## Project Structure

data/
- Raw credit data
- Processed metrics

figures/
- Model accuracy
- Calibration plots
- Stress testing figures

- Research manuscript

src/

- Feature engineering
- Modeling
- Evaluation

## Models

- Logistic Regression
- Random Forest
- Gradient Boosting

## Dataset

UCI Default of Credit Card Clients

## Research Focus

- Calibration
- Subgroup analysis
- Economic stress testing
- Decision risk
